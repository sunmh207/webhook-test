# webhook-test

[中文](./README.md) | English

This is a demo project for demonstrating webhook functionality.

## Features

- **Pascal's Triangle Generator** (`yanghui.py`): Generate and print Pascal's triangle with specified number of rows
- **Basic Utility Functions** (`test_python.py`): Contains greeting, addition, Fibonacci sequence and other basic functions

## File Description

| File | Description |
|------|-------------|
| `yanghui.py` | Pascal's triangle generation and printing |
| `test_python.py` | Basic function examples |

## Usage

### Run Pascal's Triangle

```bash
python yanghui.py
```

### Run Test Functions

```bash
python test_python.py
```

## Requirements

- Python 3.x
