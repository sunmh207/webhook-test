# webhook-test

[English](./README_en.md) | [简体中文](./README.md) | 繁體中文

這是一個演示項目，用於演示 webhook 的功能。

## 功能特性

- **楊輝三角生成器** (`yanghui.py`)：生成並列印指定行數的楊輝三角
- **基礎工具函數** (`test_python.py`)：包含問候、加法、斐波那契數列等基礎函數

## 文件說明

| 文件 | 描述 |
|------|------|
| `yanghui.py` | 楊輝三角生成與列印 |
| `test_python.py` | 基礎函數示例 |

## 使用方法

### 執行楊輝三角

```bash
python yanghui.py
```

### 執行測試函數

```bash
python test_python.py
```

## 依賴

- Python 3.x
